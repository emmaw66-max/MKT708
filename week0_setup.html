Before the course begins

Week 0 · Environment Setup

Get Python, the analytics libraries, Jupyter, and the dataset ready. Finish this before the course begins.

Goal of this page: By the end you will be able to open a Jupyter notebook, run import pandas without errors, and see the course dataset load. That's the whole bar for setup — nothing more.

Part A — Install Python

Install a current version of Python 3. The latest stable release (Python 3.14, any 3.14.x) is recommended, but anything 3.12 or newer works fine for this course. Pick your operating system:

🪟 Windows
🍎 macOS
1

Download the installer

Go to python.org/downloads/windows and download the latest Windows installer (64-bit). Most computers are 64-bit; choose that one.

2

Run the installer — check the box

Critical: On the very first installer screen, check "Add python.exe to PATH" at the bottom before clicking Install. If you skip this, the commands later won't work.

Then click Install Now and accept any admin prompt. When it finishes, click Close.

3

Verify it worked

Open the Start menu, type cmd, open Command Prompt, and run:

python --version

You should see a version number like Python 3.14.x (any 3.12+ is fine). If you instead see an error or it opens the Microsoft Store, see the troubleshooting box below.

If python isn't recognized: close and reopen Command Prompt (PATH changes need a fresh window). Still failing? Re-run the installer, choose Modify, and make sure "Add to PATH" is checked. As a fallback, try py --version.
1

Download the installer

Go to python.org/downloads/macos and download the latest macOS 64-bit universal2 installer for Python 3.

2

Run the .pkg installer

Open the downloaded .pkg file and click through Continue → Agree → Install. Enter your Mac password when prompted.

3

Verify it worked

Open Terminal (press Cmd+Space, type "Terminal", Enter) and run:

python3 --version

You should see a version number like Python 3.14.x (any 3.12+ is fine).

On macOS, always type python3 and pip3 (not python/pip). The plain commands may point to an old system Python.

Part B — Install the analytics libraries

The course needs five packages: pandas, numpy, matplotlib, seaborn, and jupyter. Install all of them in one command.

1

Run the install command

In the same Command Prompt / Terminal window:

# Windows
python -m pip install --upgrade pip
python -m pip install pandas numpy matplotlib seaborn jupyter openpyxl
# macOS
python3 -m pip install --upgrade pip
python3 -m pip install pandas numpy matplotlib seaborn jupyter openpyxl

This downloads a few hundred megabytes and takes 1–3 minutes. (openpyxl is a small extra helper for reading spreadsheet files; it's harmless to include.)

2

Confirm the libraries import

python -c "import pandas, numpy, matplotlib, seaborn; print('All good:', pandas.__version__)"

(macOS: use python3.) If it prints a version number with no error, you're done with installation.

That's the hard part over. Everything from here is just opening notebooks and writing analysis.

Part C — Set up your project folder + Jupyter

1

Make a course folder

Create a folder named MKT708 somewhere easy to find (e.g., Documents). You'll keep your notebooks and the dataset here all term.

2

Launch Jupyter Notebook

In Command Prompt / Terminal, move into your folder and start Jupyter:

# Windows example
cd %USERPROFILE%\Documents\MKT708
jupyter notebook
# macOS example
cd ~/Documents/MKT708
jupyter notebook

Your browser opens a file explorer. Click New → Notebook (choose the Python 3 kernel if prompted) to create a notebook.

3

Run your first cell

Type this into the first cell and press Shift+Enter:

import pandas as pd
print("Hello, MKT 708 —", pd.__version__)

If you see the message print below the cell, your environment is fully working.

Part D — Get the course dataset

1

Download the file from Blackboard

The dataset is provided for you on Blackboard. Download online_retail_II.csv — about 1.07 million rows of UK gift-retailer transactions, Dec 2009 – Dec 2011. You do not need to download it from any external site or create any outside account.

2

Put it in your course folder

Move online_retail_II.csv into the same MKT708 folder where your notebook lives. That keeps the file path simple in your code.

Don't open the CSV in Excel. Excel can quietly truncate the file (it caps at ~1,048,576 rows) and add blank columns when re-saved. Leave the CSV as-is and read it straight into Python.
3

Quick load test

import pandas as pd
df = pd.read_csv("online_retail_II.csv")
print(df.shape)
df.head()

You should see a shape around (1067371, 8) and the first five rows. You will reuse this exact dataset for every assignment in the course.

Column names to expect: Invoice, StockCode, Description, Quantity, InvoiceDate, Price, Customer ID, Country. Note the space in Customer ID — you'll reference it as df["Customer ID"].

Setup checklist

  • Python installed (3.12 or newer); python --version confirms it
  • pandas, numpy, matplotlib, seaborn, jupyter installed without errors
  • MKT708 project folder created
  • Jupyter Notebook opens and runs a cell
  • online_retail_II.csv downloaded into the folder and loads to ~1.07M rows
Done? You're ready for Week 1. Its guide opens in Blackboard when the course begins.
