Part 1 — Clean the data (document every step)

1

Track counts with a small log

log = []
def record(label, frame):
    log.append({"step": label, "rows": len(frame)})
    print(f"{label:<35} rows = {len(frame):,}")

record("0. Raw data", df)
2

Remove cancellations

Cancelled orders have an Invoice number starting with the letter "C". Remove them.

df["Invoice"] = df["Invoice"].astype(str)
df = df[~df["Invoice"].str.startswith("C")]
record("1. Removed cancellations (C)", df)
3

Remove records without a Customer ID

RFM and CLV are per-customer, so a transaction with no customer is unusable for our analysis.

df = df[df["Customer ID"].notnull()]
record("2. Removed missing Customer ID", df)
4

Remove zero / negative price and quantity

df = df[df["Price"] > 0]
record("3. Removed Price <= 0", df)

df = df[df["Quantity"] > 0]
record("4. Removed Quantity <= 0", df)
5

(Optional but recommended) drop exact duplicate rows

df = df.drop_duplicates()
record("5. Removed duplicate rows", df)
6

Build a line-revenue column

df["Customer ID"] = df["Customer ID"].astype(int)
df["LineRevenue"] = df["Quantity"] * df["Price"]
df.head()
Why cast Customer ID? Because the raw file has missing IDs, pandas reads the column as a float (you'll see 12346.0). After dropping the missing rows above, .astype(int) makes it a clean whole number. (If you ever need to keep IDs and allow missing values, use the nullable .astype("Int64") instead.)
7

Show the cleaning impact table

impact = pd.DataFrame(log)
impact["rows_removed"] = impact["rows"].shift(1) - impact["rows"]
impact["pct_of_raw"] = (impact["rows"] / impact["rows"].iloc[0] * 100).round(1)
impact
Write 1–2 sentences per cleaning step in a Markdown cell: what you removed, how many rows, and why it's justified. That narrative is the core of the grade.

Part 2 — Customer-level summary table

Aggregate the cleaned transactions up to one row per customer: number of orders, total revenue, and active date range.

customer = df.groupby("Customer ID").agg(
    orders       = ("Invoice", "nunique"),
    items        = ("Quantity", "sum"),
    revenue      = ("LineRevenue", "sum"),
    first_order  = ("InvoiceDate", "min"),
    last_order   = ("InvoiceDate", "max"),
).reset_index()

customer["avg_order_value"] = (customer["revenue"] / customer["orders"]).round(2)
customer["tenure_days"] = (customer["last_order"] - customer["first_order"]).dt.days

print("Customers:", len(customer))
customer.describe()
customer.sort_values("revenue", ascending=False).head(10)

Viz 5 — Revenue concentration (the 80/20 check)

rev_sorted = customer["revenue"].sort_values(ascending=False).reset_index(drop=True)
cum_share = rev_sorted.cumsum() / rev_sorted.sum()
plt.figure(figsize=(8,5))
plt.plot(np.arange(1, len(cum_share)+1) / len(cum_share), cum_share)
plt.title("Revenue Concentration: Cumulative Share of Revenue by Customer")
plt.xlabel("Share of customers (ranked high to low)")
plt.ylabel("Cumulative share of revenue")
plt.show()

top20 = cum_share.iloc[int(len(cum_share)*0.2)]
print(f"Top 20% of customers drive {top20:.0%} of revenue")
