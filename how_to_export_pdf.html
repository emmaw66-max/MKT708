How to Export Your Notebook to PDF

Every Applied Assignment is submitted as a PDF (or HTML), not the raw .ipynb. Here are three ways to export — use whichever works on your machine.

Do this first — every time: before exporting, run the whole notebook so your code, outputs, and charts are visible. Top menu → Run → Run All Cells (or Kernel → Restart Kernel and Run All Cells). A PDF made from an un-run notebook will be missing its charts.

Method 1 — Recommended  Print to PDF from your browser

This is the most reliable method and works on any computer — no extra software needed.

1
Export to HTML first. In Jupyter: File → Save and Export Notebook As… → HTML. An .html file downloads.
2
Open that HTML file (double-click it — it opens in your web browser).
3
Press Ctrl+P (Windows) or Cmd+P (Mac) to open Print.
4
Under Destination / Printer, choose “Save as PDF”, then click Save. Done.
Why this one: it never fails for missing software, and the PDF looks exactly like your notebook — code, outputs, and every chart.

Method 2  Direct PDF export from Jupyter

1
In Jupyter, top menu: File → Save and Export Notebook As… → PDF.
2
If a PDF downloads — great, you're done.
Heads up: this option needs an extra typesetting system (LaTeX) installed, and on many laptops it errors out. If you get an error about “xelatex” or “nbconvert failed,” don't fight it — just use Method 1 above.

Method 3  WebPDF (if you see the option)

Some Jupyter installs offer File → Save and Export Notebook As… → Webpdf. If it's there, it produces a clean PDF directly (it uses a built-in browser engine). If it asks to download a component the first time, allow it. If it's not listed or fails, fall back to Method 1.

Before you submit

Check your PDF: open it and confirm every chart is visible and not cut off, your written interpretations are included, and the file isn't blank. Then rename it to the required format, e.g.:
Smith_Jane_MKT708_AA1.pdf

Upload to Blackboard and confirm the “Successfully Submitted” mark.
